## Overview

**ReminderTask** is a practice iOS app developed during the UTS *Application Development in the iOS Environment* course. The focus was on exploring native iOS frameworks and delivering a functional app.

## Features

- Set and manage reminders with local notification support
- Task prioritisation to organise to-dos by importance
- Clean native iOS UI using UIKit / SwiftUI

## Tech Stack

- **Language**: Swift
- **Platform**: iOS
- **Tools**: Xcode, iOS Simulator

## Outcome

Presented the project in a 5-minute demo, demonstrating proficiency with iOS frameworks and the Swift development lifecycle. Repository maintained on GitHub.
