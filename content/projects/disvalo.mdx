## Overview

**Disvalo** is a personal project — a Discord bot built to track and display **Valorant** player statistics without leaving the Discord app. Built out of a love for gaming and a desire to put JavaScript skills to practical use.

## Features

- Look up Valorant player rank and match history via Discord slash commands
- Displays KDA, win rate, and recent match stats
- Live data fetched from the Valorant API

## Tech Stack

- **Language**: JavaScript (Node.js)
- **Library**: discord.js
- **API**: Valorant / Henrik Dev API

## Why I Built It

I wanted a side project that combined two things I enjoy — gaming and coding. It was a great way to get hands-on with API integration, async JavaScript, and bot deployment.
