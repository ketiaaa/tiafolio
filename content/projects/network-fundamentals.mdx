## Overview

Completed as part of the UTS College *Networking Fundamentals* subject, this project documents and demonstrates understanding of foundational networking concepts through practical exercises and a web-based report.

## Topics Covered

- TCP/IP model and OSI layers
- IP addressing and subnetting
- DNS and DHCP configuration
- Basic router and switch setup
- Network troubleshooting methodology

## Tech Stack

- **Presented as**: HTML/CSS web report
- **Tools**: Cisco Packet Tracer, terminal/CLI

## Relevance

This foundational knowledge directly supported my later work as a **System & Network Analyst** at the National Bank of Cambodia, where I applied these concepts in a live banking infrastructure environment.
