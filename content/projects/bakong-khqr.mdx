## Overview

During my time at the **National Bank of Cambodia**, I worked on IT support for the **Bakong KHQR** system — Cambodia's national QR code payment standard built on blockchain infrastructure.

## What is Bakong?

Bakong is the NBC's blockchain-based payment system that allows real-time interoperability between participating banks and financial institutions using a standardised QR code format (KHQR).

## My Role

- Provided IT support for connectivity between financial institution systems and the Bakong network
- Troubleshot integration issues between bank endpoints and the NBC's central ledger
- Assisted in verifying KHQR compliance for participating institutions
- Supported in-branch and back-office IT infrastructure

## Tech Context

- **Payment Rail**: Bakong (Hyperledger Iroha-based blockchain)
- **Standard**: KHQR (EMV-compatible QR code specification)
- **Environment**: Banking network, internal IT systems
