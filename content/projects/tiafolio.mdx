## Overview

tiafolio is my personal portfolio, rebuilt using the [chronark.com](https://chronark.com) open-source structure as a foundation.

## Stack

- **Framework**: Next.js 13 (App Router)
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion — mouse-reactive card spotlight effect
- **Content**: Contentlayer + MDX for project pages
- **Font**: Cal Sans (display) + Inter (body)

## Features

- Animated particle field on the hero that reacts to mouse movement
- Glowing horizontal sweep lines on page load
- Large clip-text hero name with a slow reveal animation
- Mouse-spotlight cards on the projects page
- MDX-powered individual project pages
- Blur-on-scroll navigation bar
